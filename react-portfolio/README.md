# Selamawit Mesfin Portfolio

A modern single-page React portfolio built with Vite and Tailwind CSS.

## Features

- Midnight blue developer theme with cyan and emerald accents
- Hero, Skills, Projects, Experience, and Contact sections
- GitHub and LinkedIn links
- Focus on full-stack systems, SQL databases, React.js, and Ethiopian agriculture solutions

## Setup

1. Open the project folder in VS Code:
   `C:\Users\HP\react-portfolio`
2. Install dependencies:
   `npm install`
3. Run the development server:
   `npm run dev`

## Project Structure

- `index.html` - main HTML entry
- `src/main.jsx` - React application bootstrap
- `src/App.jsx` - portfolio single-page UI
- `src/index.css` - Tailwind and global styles
- `tailwind.config.js` - Tailwind configuration
- `vite.config.js` - Vite configuration
